# Security Policy

## Supported versions
This project is currently in MVP stage. Security fixes are applied to the latest mainline state.

## Reporting a vulnerability
Please do not open public issues for suspected vulnerabilities.

Share the report privately with:
- affected area (backend, Flutter app, infra, dependency)
- impact and exploit scenario
- reproduction steps or proof of concept
- suggested remediation if available

Maintainers will acknowledge reports, assess severity, and provide a remediation timeline.

## Disclosure
After a fix is available, coordinated disclosure is encouraged with clear upgrade guidance.
